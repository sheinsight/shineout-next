## ✨ 特性

 - 精巧且友好的 API
 - 开箱即用的高性能 React 组件
 - 接口永远向下兼容
 - 灵活的主题配置

## 🛠️ 依赖

```
react >= 16.14.0
react-dom >= 16.14.0
```
