`````
# 国际化

所有组件文案默认使用的是英文，通过设置可以使用其它语言
`````

## 内置语种

内置了两种语言：简体中文（zh-CN）和英文（en-US）两组, 默认为英文（en-US）,可以通过 setLocale 进行切换

```js
import { setLocale } from 'shineout'
setLocale('zh-CN')
```

可以通过 webpack 的 process.env 切换
```js
plugins: [
  new webpack.DefinePlugin({
    'process.env': {
      LOCALE: JSON.stringify('zh-CN'),
    },
  }),
],
```

## 自定义语种
其他语言或者部分设置，可以传入一个 Json 数据
```js
setLocale({ ok: 'yes' })
```
